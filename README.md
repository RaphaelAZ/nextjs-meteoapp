# Meteo App

Meteo App est une application météo simple pour chercher une ville, consulter les prévisions, comparer la météo entre plusieurs villes et garder quelques lieux en favori.

Vous trouverez des captures d'écrans dans le dossier > /screens/

## Lancer l'application

```bash
npm install
npm run dev
```

Puis ouvrez [http://localhost:3000](http://localhost:3000).
